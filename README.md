A little clone of the game [Floating Point](https://store.steampowered.com/app/302380/) by Suspicious Developments.

Made in [Zig](https://ziglang.org) 0.16.0 and CNLohr's [rawdraw](https://github.com/cntools/rawdraw) library.
Build with:
```
zig build
```
Needs `libX11` on Linux and `gdi32` on Windows.
